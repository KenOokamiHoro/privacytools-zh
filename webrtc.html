WebRTC IP Leak VPN / Tor IP Test

WebRTC 是一个通讯协议，即便通过 VPN 上网，其使用的 JavaScript 可能会泄露你真实的IP地址。有些软件，如 NoScript 可以阻止这个漏洞。但为了安全起见，也许直接关闭这个协议也是不错的作法。这里可以测试你的网页浏览器是否受到 WebRTC 漏洞的影响。

这个示例秘密地把请求送到 STUN 服务器，它会记录下你的对外请求。这些对外的动作并不会显示在开发者控制台，也不能利用浏览器扩展（例如 AdBlock, Ghostery等）对其进行封锁。

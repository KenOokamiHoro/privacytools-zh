PrivacyTools 网络隐私工具

你正被盯着。私人和国家支持的组织正在监控与记录你的在线活动。

At PrivacyTools 提供一些知识与工具以保护个人隐私抵抗全球大规模监控，并推动持相近隐私关切的个人，一同参与社群交流讨论，学习如何保护个人网络数据安全。隐路隐私工具为本组织的内核组成，通过它的研究向社群推荐各式软件与服务方案。

公开透明是我们坚持的价值，也是区隔本站和其它隐私社群不同之处。主站信息的编辑变动与所推荐的产品都是先通过密集的交流，如 GitHubissue tracker，由贡献者提交草稿请求 作进一步的讨论。有心者可以回头查看自 2015 年成立以来所走过的每一笔提交修改记录。下方列出主要内核core团队成员，他们为最终的编写定稿与服务维护负起责任。事实上，本站背后由上百位 贡献者与查找者努力确保所推荐的东西是可靠与让人信赖的。

PrivacyTools 为非营利组织，我们不会收取厂商报酬来推荐产品，虽然类似的置入行销手法在其它地方很普徧，那种操作法不幸地伤害了读者对各家评比网站的信任感。PrivacyTools 采取不同策略，我们的研究是独立客观，绝不收取金钱报酬后修改原有的推荐或评鉴意见。本站财务完全依靠社群的捐款人 与 赞助者，且依照美国税法 Open Collective Foundation 501(c)(3) 相关规定。作为在美国登记的非营利组织，我们有遵法义务只能将基金花在符合宗旨的用途，即推广隐私教育和提倡使手尊重隐私的工具服务。本站是一个公共资源库，不是利润机器，所以相关的财务交易都妥善记录且公布于 opencollective.com/privacytoolsio供查找阅览。

我们严肃认真地经营各式 在线服务，要求所有的参与者必须遵守良好之行为规范 Code of Conduct。 若有疑问或欲申报不当使用，请参考行为规范的Enforcement 落实篇。

值得一提

  • Cryptomator - 免费客户端的云端档案 AES 加密工具。其为开源软体，没有后门也无须注册。
  • miniLock - Google Chrome / Chromium 浏览器的外挂，可用一组密码来加密档案，其开发者来自 Cryptocat。
  • AES Crypt - 使用强力 256-位元的加密演算法，AES Crypt 可以安全地保护最敏感的重要资料档案，可使用在Windows, macOS, Linux 与 Android 平台上。
  • DiskCryptor - 可为 Windows 系统作整颗硬碟或分割磁区的加密系统，包括加密某个分割磁区或是作业系统所安装的硬碟。
  • Linux Unified Key Setup (LUKS) - 使用 dm-crypt 为磁碟加密骨干，可为 Linux 全磁区加密系统。Ubuntu 作业环境提供其为默认设定，也有 Windows 与其它的 Linux 版本可用。

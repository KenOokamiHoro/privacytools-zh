值得一提

  • Replicant {% include badge.html color="info" text="AOSP" %} - 基于 Android 的自由操作系统，以完全替代 Android 中的私有组件为目标。
  • OmniROM {% include badge.html color="info" text="AOSP" %} {% include badge.html color="warning" icon="far fa-question-circle" text="部分自由" tooltip="此软件可能依赖一些私有软件。" %} - 基于 AOSP 的另一个开放源代码的 Android 系统。
  • MicroG {% include badge.html color="info" text="额外组件" %} {% include badge.html color="warning" icon="far fa-question-circle" text="部分自由" tooltip="此软件可能依赖一些私有软件。" %} - 以自由和开放源代码的实现替代私有的 Google Play 服务的计划。microG 也维护包含 microG 的 LineageOS 构建， Lineage for microG。
